# S1000D React Demo

[English](README.md) | 简体中文

React + Tiptap 的 S1000D 可视化 demo。

## 运行

```bash
npm install
npm run dev:demo:s1000d
npm run dev --workspace s1000d-react-demo
```

## E2E

```bash
npm run test:e2e:s1000d
```

这个 demo 只使用公开的 S1000D 包入口：

- `prosemirror-html-table-s1000d`
- `prosemirror-html-table-s1000d/tiptap`
- `prosemirror-html-table-s1000d/clipboard`
- `prosemirror-html-table-s1000d/renderer`
