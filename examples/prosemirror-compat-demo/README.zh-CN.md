# Pure ProseMirror Compat Demo

[English](README.md) | 简体中文

最小化的纯 ProseMirror 兼容性 demo，用来验证 `prosemirror-html-table` 核心能力。

```bash
npm run dev --workspace prosemirror-compat-demo
npm run build --workspace prosemirror-compat-demo
```

这个 demo 使用 `tableEditing()`、`CellSelection.content()` 和 `officialCompat`，不依赖 Tiptap。
