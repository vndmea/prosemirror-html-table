# Pure ProseMirror Compat Demo

Minimal ProseMirror demo for the core `prosemirror-html-table` package.

```bash
npm run dev --workspace prosemirror-compat-demo
npm run build --workspace prosemirror-compat-demo
```

The demo uses `tableEditing()`, `CellSelection.content()`, and `officialCompat` helpers without Tiptap.
