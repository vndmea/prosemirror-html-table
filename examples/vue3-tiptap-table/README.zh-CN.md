# Vue 3 + Tiptap v3 HTML Table Demo

[English](README.md) | 简体中文

这个示例演示了 `tiptap-html-table` 在 Vue 3 + Tiptap v3 应用中的用法。

## Playground

在线：

[Playground](https://vndmea.github.io/prosemirror-html-table/)

本地：

```bash
npm install
npm run dev --workspace vue3-tiptap-table-demo
```

它包含完整 HTML table 结构，以及行列手柄、上下文菜单、resize 与扩展控件、selection overlay 和表级工具栏。
